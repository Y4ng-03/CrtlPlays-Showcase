Bienvenido a **CtrlPlays**, una solución de comercio electrónico robusta y personalizada, diseñada específicamente para la venta de licencias de videojuegos y consolas. Este proyecto demuestra una arquitectura limpia, un diseño enfocado en la experiencia de usuario (UX) y una integración fluida con sistemas de mensajería.

## 🚀 Funcionalidades Clave

- **🛒 Sistema de Carrito Dinámico:** Gestión en tiempo real de productos, cantidades y licencias (Primarias/Secundarias).
- **📱 Optimización Móvil:** Interfaz 100% responsiva diseñada para una navegación fluida en cualquier dispositivo.
- **💬 Integración con WhatsApp:** Generación automática de tickets de pedido detallados enviados directamente al vendedor.
- **🔐 Seguridad Avanzada:** Protección contra Inyección SQL, validación de datos en servidor y manejo seguro de sesiones.
- **🛠️ Panel de Administración:** Gestión completa de inventario, actualización de precios y control de stock.

---

## 🛠️ Stack Tecnológico

*   **Backend:** PHP (Arquitectura Modular)
*   **Frontend:** HTML5, CSS3 (Vanilla para máximo rendimiento), JavaScript ES6+
*   **Base de Datos:** MySQL (Relacional)
*   **Servidor:** Entorno XAMPP / Apache
